System requirement and dependencies


>>iTAK server


iTAK server has been tested under firefox, IE, and Safari. If it's incompatible with your browser, please feel free to contact us.

>>iTAK standalone program


  • Linux (required)
  • Perl version 5.10.0 or higher (required). Perl was installed by default on most Linux systems
  • BioPerl version 1.006 or higher (required). Please check http://www.bioperl.org and wiki/Installing_BioPerl for more details on installation of BioPerl.
  • HMMER3 (required). Provided in iTAK.
  • 2.0 GB free hard disk space.

Installation


>>iTAK standalone program


Installation of iTAK is straightforward. First download the latest version of iTAK for your system and uncompress the downloaded file. It will generate a folder named "iTAK-1.0.x32" on a 32-bit machine or "iTAK-1.0.x64" on a 64-bit machine (we call this folder "iTAK home folder"). iTAK home folder includes three subfolders, a "bin" folder containing the HMMER3 executable, a "database" folder containing the domain database files and a "doc" folder containing the program documentation files. The home folder also contains a perl script, iTAK.pl, which is the core script to run the whole iTAK pipeline. Next download the formatted domain database files from the download page. Uncompress and move the database files to the "database" folder.


Running iTAK


>>standalone version


Quick Start

  1. Put the protein or nucleotide sequence file in FASTA format in iTAK home folder
  2. Go to iTAK home folder and run iTAK with the following command (assuming the input file name is input_seq)

    3. shell$ perl iTAK.pl -i input_seq

  3. The program will generate an output folder named input_seq_output which contains all the output files. See below for the description of the output files.

Parameters

-i[String]Name of the input sequence file in FASTA format (required)
-s[String]Type of input sequence. 'p' for protein sequences | 'n' for nucleotide sequences. (default = p)
-m[String]Type of analysis ('t' for TF identification | 'p' for PK identification | 'b' for both) (default = b)
-a[Integer]number of CPUs used for hmmscan (default = 1)
-o[String]Name of the output directory (default = "input file name" + "_output")


iTAK output


>>iTAK server


The output of iTAK web server contains tables which list the identified transcription factors, transcription regulators, and protein kinases, as well as their corresponding families (see figure below). The detailed alignments of each protein sequence against protein domains are also provided.



An example output is provided here.

>>iTAK standalone program


Six files will be generated in the output directory - "input_seq_output" (assuming the input file name is "input_seq").

  1. input_seq_tf_seq: sequences of all identified transcription factors (FASTA format).
  2. input_seq_tf_family: classification of all identified transcription factors. A tab-delimited txt file containing sequence IDs and their corresponding transcription factor families.
  3. input_seq_tf_align: A tab-delimited txt file containing alignments of all identified transcription factors to the protein domain database (from hmmscan analysis).
  4. input_seq_pkseq: sequences of all identified protein kinases (FASTA format).
  5. input_seq_pkcat: classification of all identified protein kinases. A tab-delimited txt file containing sequence IDs and their corresponding protein kinase families.
  6. input_seq_pkaln: A tab-delimited txt file containing alignments of all identified transcription factors to the protein domain database (from hmmscan analysis).

Performance


We run iTAK on a single CPU on a 32-bit laptop with Intel Core2 Duo P9400 @ 2.40GHz and 4GB memory. It took aout 3.5 hours to identify both transcription factors and protein kinases from 33,410 Arabidopsis protein sequences (TAIR9 release).
