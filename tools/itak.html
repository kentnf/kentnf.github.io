System requirement and dependencies


>>iTAK server


iTAK server has been tested under firefox, IE, and Safari. If it's incompatible with your browser, please feel free to contact us.

>>iTAK standalone program


  • 64-bit Linux system
  • Perl version 5.10.0 or higher (required). Perl was installed by default on most Linux systems
  • BioPerl version 1.006 or higher (required). Please check http://www.bioperl.org and wiki/Installing_BioPerl for more details on installation of BioPerl.
  • HMMER3 (required). Provided in iTAK.

Installation


>>iTAK standalone program


Installation of iTAK is more straightforward than previous. Everything you need is to download the lastest version of iTAK and uncompress it.

shell$ tar -xzvf iTAK-1.5.tar.gz

It will generate a folder named "iTAK-1.x" and we call this folder "iTAK home folder". iTAK home folder includes three subfolders, a "bin" folder containing the HMMER3 executable, a "database" folder containing the domain database files and a "doc" folder containing the program documentation files. The home folder also contains a perl script, iTAK.pl, which is the core script to run the whole iTAK pipeline.

After that, please run test sequences to initialize databases.

shell$ cd iTAK-1.5
shell$ perl iTAK.pl test_seq


For 32-bit user
Only 64-bit version will be provide from iTAK v1.5. The only difference between 64-bit and 32-bit iTAK is recruitment of different versions of HMMER. To run iTAK on 32-bit machine, please replace the HMMER with ia32 version.


Running iTAK


>>standalone version


Quick Start

  1. Put the protein or nucleotide sequence file in FASTA format in iTAK home folder
  2. Go to iTAK home folder and run iTAK with the following command (assuming the input file name is input_seq)

    3. shell$ perl iTAK.pl input_seq

  3. The program will generate an output folder named input_seq_output which contains all the output files. See below for the description of the output files.

Parameters

-f[String]Frame usage of nucleotie translation, could be:
6: 6 frame translation (default)
3F: 3 frame translation with forward strand
3R: 3 frame translation with reverse strand
-a[Integer]number of CPUs used for hmmscan (default = 1)
-o[String]Name of the output directory (default = "input file name" + "_output")


obsoleting parameters
-i[String]Name of the input sequence file in FASTA format (required)
-s[String]Type of input sequence. 'p' for protein sequences | 'n' for nucleotide sequences. (default = p)
-m[String]Type of analysis ('t' for TF identification | 'p' for PK identification | 'b' for both) (default = b)


iTAK output


>>iTAK server


The output of iTAK web server contains tables which list the identified transcription factors, transcription regulators, and protein kinases, as well as their corresponding families (see figure below). The detailed alignments of each protein sequence against protein domains are also provided.



An example output is provided here.

>>iTAK standalone program


Eight files will be generated in the output directory - "input_seq_output" (assuming the input file name is "input_seq").

  1. tf_sequence.fasta: sequences of all identified transcription factors (FASTA format).
  2. tf_classification.txt: classification of all identified transcription factors. A tab-delimited txt file containing sequence IDs and their corresponding transcription factor families.
  3. tf_alignment.txt: A tab-delimited txt file containing alignments of all identified transcription factors to the protein domain database (from hmmscan analysis).
  4. pk_sequence.fasta: sequences of all identified protein kinases (FASTA format).
  5. PPC_classification.txt: classification of all identified protein kinases. A tab-delimited txt file containing sequence IDs and their corresponding protein kinase families.
  6. PPC_alignment.txt: A tab-delimited txt file containing alignments of all identified protein kinases to the protein domain database (from hmmscan analysis).
  7. Shiu_classification.txt: classification of all identified protein kinases. A tab-delimited txt file containing sequence IDs and their corresponding protein kinase families.
  8. Shiu_alignment.txt: A tab-delimited txt file containing alignments of all identified protein kinases to the protein domain database (from hmmscan analysis).


Performance


We run iTAK on a single CPU on a 64-bit laptop with i5-3320M CPU @ 2.60GHz. It took aout 27 minutes to identify both transcription factors and protein kinases from 33,410 Arabidopsis protein sequences (TAIR10 release).
